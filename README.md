# quiz_cultura_general
Juego de preguntas de cultura general en las cuales hay dos tipos de preguntas en las cuales son de opción multiple y de verdadero y falso, en las cuales son 4 preguntas de verdadero y falso y 4 preguntas de opcion multiple en la cual solamente una opcion correcta 
#Indicaciones
En las cuales para poder ejecutar este juego se necesita tener visual code y tener instalando Python 3.14.6final.0 para poder meter lo que son los codígos
* Se debe de descargar el formato "preguntas.json"
