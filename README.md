# quiz_cultura_general
Juego de preguntas de cultura general en las cuales hay un solo tipo de pregunts en las cuales son de opción multiple, en las cuales solamente una opcion es la correcta, que son preguntas de cultura general 
# Indicaciones
En las cuales para poder ejecutar este juego se necesita tener visual code y tener instalando Python 3.14.6final.0 para poder meter lo que son los codígos
* Se necesita descragar la carpeta de PROYECTO_QUIZ
